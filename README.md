# 📊 Grafana Dashboards

> Coleção de dashboards Grafana desenvolvidos para monitoramento de infraestrutura utilizando Zabbix.

---

## 🚀 Objetivo

Este repositório reúne dashboards criados para monitoramento de ambientes corporativos, com foco em disponibilidade, desempenho e observabilidade da infraestrutura.

Os projetos publicados aqui utilizam dados anonimizados e têm como objetivo compartilhar conhecimento e boas práticas com a comunidade.

## 📊 Dashboards

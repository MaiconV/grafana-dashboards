# 🖥 NOC Dashboard

## 📖 Sobre

Dashboard desenvolvido para monitoramento centralizado da infraestrutura de TI utilizando Grafana e Zabbix.

O objetivo é fornecer uma visão rápida da saúde do ambiente, permitindo identificar indisponibilidades e degradações de serviços em tempo real.

---

## Recursos

- Monitoramento de Firewalls
- Links de Internet
- Latência
- Jitter
- Perda de Pacotes
- Disponibilidade
- VPNs
- Uso de CPU
- Uso de Memória

---

## Tecnologias

- Grafana
- Zabbix
- ICMP
- SNMP
- FortiGate
